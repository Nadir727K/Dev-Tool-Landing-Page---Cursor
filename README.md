# 🚀 Cursor Landing Page Clone

A modern SaaS-style landing page clone inspired by Cursor (AI-powered code editor). Built using HTML and CSS with a focus on layout design and real-world UI structure.

---

## 📸 Preview
![Preview](assets/preview.png)

---

## 🧠 About the Project

This project replicates a modern SaaS landing page UI similar to Cursor. It includes multiple sections commonly found in product websites such as hero, trusted-by section, changelog, team section, and footer.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid)  

---

---

## ✨ Features

### 🎯 Navbar
- Logo on left
- Navigation links in center
- Action buttons on right


### ⚡ Hero Section
- Large headline with CTA button
- Product screenshot display

### 🏢 Trusted By Section
- Company logos in card layout

### 📦 Feature Section
- Product showcase image

### 📜 Changelog Section
- Update cards in grid layout

### 👨‍💻 Team Section
- Split layout (text + image)
- Join us CTA button

### 🔚 Footer
- Multi-column link structure
- Social + settings section
